# Security Policy

## Supported Images

This repository provides base container images intended for reuse in CI/CD pipelines and application images.
Only the most recent versions of images are actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability:

- **Do not** open a public issue
- Report it privately via a GitHub security advisory or direct contact

Please include:
- Image name and tag
- Description of the issue
- Steps to reproduce if applicable

## Response Expectations

Tomshley LLC will:
- Acknowledge receipt of the report
- Assess the impact
- Publish fixes or mitigations as appropriate

Timelines may vary depending on severity and scope.
